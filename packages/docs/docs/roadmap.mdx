## Releases

### 0.0.1 – MVP

- Working documentation (with small hiccups on mobile browsers).
- Full keyboard accessibility (with occasional bugs).
- Full dark mode support.
- New components:
  - Forms: `Checkbox`, `Label`, `Select`, `Switch`, `TextInput`.
  - Navigation: `MenuButton`, `Modal`, `Pagination`, `Popover`.
  - Miscellaneous: `Button`, `HorizontalLine`, `Pill`, `Spinner`, `Table`, `Tag`, `Tooltip`.

### 0.0.2 – Proper bundling

- Changed bundler config to publish separate files instead of single CSS file to support better tree-shaking.
- New components: `Breadcrumbs`.

> We are here now.

### 0.0.3 – Animations

- Add animations and transitions.
- New components: `Accordion`.

### 0.0.4 – Forms

- New components: `RadioGroup`, `CheckboxGroup`.
- Error message and description for `TextInput`.
- Improved [Formik guide](/docs/guides/formik) and added examples of checkbox and radio.

### 0.0.5

- More examples.
- New components (a subset of those):
  - Forms: `Radio`, `RadioGroup`, `CheckboxGroup`.
  - Navigation: `TabList`.
  - Inputs: `SearchField`, `ComboBox`.
  - Miscellaneous: `ProgressBar`, `Slider`.

### 0.0.6

- Testing of screen reader accessibility.

### 0.0.7

- Test pipelines for ensuring quality.

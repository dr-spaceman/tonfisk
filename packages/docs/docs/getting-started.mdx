## Installation

```bash
yarn add TODO_LIB_NAME
```

## Setup

### Provider

`<Provider />` is absolutely required for the library to run. There should be only one provider in the component tree.

```jsx
import { Provider } from "TODO_LIB_NAME";

export default function App() {
  return <Provider>{/* Your app */}</Provider>;
}
```

### Styling

Set up `vanilla-extract` according to your bundler settings: [guide](https://vanilla-extract.style/documentation/setup/).

### Dark theme

In order to avoid flash of white background if user is supposed to see dark mode, copy the following file: [noflash.js](/noflash.js) and save it as `noflash.js` file to `/public` directory of your repository.

If you are using Next.js, it will be `/public`.

Next, if you have your own custom `index.html`, add `<script src="/noflash.js"></script>` inside `<body>`.

If you are using Next.js, you will need `_document.js`:

```js
import Document, { Head, Html, Main, NextScript } from "next/document";

class MyDocument extends Document {
  static async getInitialProps(ctx) {
    const initialProps = await Document.getInitialProps(ctx);
    return { ...initialProps };
  }

  render() {
    return (
      <Html>
        <Head />
        <body>
          <script src="/noflash.js" />
          {/* ^ here */}
          <Main />
          <NextScript />
        </body>
      </Html>
    );
  }
}

export default MyDocument;
```

## Usage

Simple snippet showing how to get started:

```jsx
import { Button } from "TODO_LIB_NAME";

export default function App() {
  return (
    <div>
      <Button onPress={() => console.log("Hej!")}>Press me</Button>
    </div>
  );
}
```

## Flow of writing

Most of the code can be composed using ready components. But often there will be situations custom enough that it won't make sense to have ready-made components in the design system. That's why TODO_LIB_NAME leverages `vanilla-extract` and fully type safe styling.

Example code can look like this:

```jsx
<div
  className={atoms({
    display: "flex",
    gap: "m",
    padding: "m",
    boxShadow: {
      lightMode: "regularBorder",
      darkMode: "darkBorder",
    },
    borderRadius: "8px",
  })}
>
  <p>Hello there.</p>
  <Button
    onPress={() => {
      console.log("Hej!");
    }}
  >
    Press me
  </Button>
</div>
```

To learn more, check out [documentation](https://vanilla-extract.style/documentation) of `vanilla-extract`.
